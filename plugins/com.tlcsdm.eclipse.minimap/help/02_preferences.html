The Minimap Preference Page

You can configure various preferences to trim Minimap to your liking. The preference section can be reached via Window - Preferences - Minimap. Minimap has these settings:

The minimap includes a "fog" effect to help you focus on the currently visible area in the editor window. Portions of the file that are outside of the visible viewport are covered with a semi-transparent overlay.

Fog Color

This setting defines the color of the fog applied to the non-visible regions in the minimap. The color can be customized to match your theme or preferences.

Fog Transparency

This setting controls how opaque or transparent the fog overlay appears. The value ranges from 0 (fully transparent) to 255 (fully opaque).

  • Lower values make the fog less visible
  • Higher values increase the visual contrast between active and inactive areas
