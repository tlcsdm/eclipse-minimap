Introduction

Eclipse plugin provides awesome way to navigate your source code.

The Eclipse Minimap plugin adds a compact overview of your code (a "Minimap") on the right side of the editor window, similar to the minimap feature found in popular editors like Visual Studio Code.

This allows users to quickly scroll through and understand the structure of long source files without losing context. It improves code navigation and boosts productivity, especially in large files.

Usage

The minimap is automatically enabled for supported editors when the plugin is installed. To see the minimap:

  • Open any supported file (e.g., Java, XML, C/C++, or text files)
    • >
  • Open Window - Show View - Other... - Plug-in Development - Minimap
  • The minimap view will be appear and then placed to the right side of the editor
  • Click or scroll within the minimap to quickly navigate to specific sections of the file
